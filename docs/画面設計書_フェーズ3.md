# ユナメイト 画面設計書（フェーズ3）

本書はフェーズ3（ドラフトモジュール化・ルーム型ドラフト）に必要な画面を定義する。
マッチングを経由しない「ルーム作成/参加」と「席決定」を中心に構成する。

---

## 1. 画面一覧

- P3-01 ログイン/登録（P1同等）
- P3-02 ルーム一覧/参加
- P3-03 ルーム作成
- P3-04 ルーム待機（席選択）
- P3-05 ドラフト（BAN/PICK）
- P3-06 ドラフト結果
- P3-07 ルーム完了
- P3-08 設定/プロフィール
- P3-09 エラー/メンテ

---

## 2. 画面詳細

### P3-01 ログイン/登録
- 目的: ユーザー認証
- 主なUI: フェーズ1同等

### P3-02 ルーム一覧/参加
- 目的: 既存ルームへの参加
- 主なUI:
  - 参加コード入力欄
  - 参加ボタン
- 状態:
  - 入力中 / 参加中 / エラー
- アクション:
  - 参加

### P3-03 ルーム作成
- 目的: ルーム作成
- 主なUI:
  - 作成ボタン
  - 生成された参加コード表示
- 状態:
  - 作成中 / 作成完了
- アクション:
  - ルーム作成

### P3-04 ルーム待機（席選択）
- 目的: 席選択と参加者の揃い確認
- 主なUI:
  - 先攻/後攻の席表示（5席ずつ）
  - 着席ボタン
  - 参加者一覧
  - 満席状態表示
- 状態:
  - 未着席 / 着席済み / 満席 / ドラフト開始可能
- アクション:
  - 席に座る
  - 席を離れる
  - ドラフト開始（オーナー）

### P3-05 ドラフト（BAN/PICK）
- 目的: BAN/PICKを進行
- 主なUI/状態/アクション: フェーズ1同等

### P3-06 ドラフト結果
- 目的: BAN/PICK確定結果を閲覧
- 主なUI:
  - 先攻/後攻の最終ピック表示
  - BAN一覧
- 状態:
  - 表示のみ

### P3-07 ルーム完了
- 目的: ドラフト完了後の終了状態
- 主なUI:
  - 完了メッセージ
  - 再ドラフト導線（将来拡張）
- 状態:
  - 完了

### P3-08 設定/プロフィール
- 目的: 表示名や基本設定の変更
- 主なUI/アクション: フェーズ1同等

### P3-09 エラー/メンテ
- 目的: システムエラー時の案内
- 主なUI/アクション: フェーズ1同等

---

## 3. 画面遷移（概要）

- P3-01 → P3-02 / P3-03
- P3-03 → P3-04
- P3-02 → P3-04
- P3-04 → P3-05 → P3-06 → P3-07
- 例外: エラー時は P3-09
