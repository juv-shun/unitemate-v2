# ユナメイト 画面設計書（フェーズ2）

本書はフェーズ2（ランクマッチ拡張）に必要な画面を定義する。
フェーズ1の画面に加え、結果入力・ランキング・レーティング関連を追加する。

---

## 1. 画面一覧

- P2-01 ログイン/登録（P1同等）
- P2-02 ホーム（インキュー/ステータス）
- P2-03 マッチ成立
- P2-04 ドラフト（BAN/PICK）
- P2-05 ロビーID入力/共有
- P2-06 試合結果入力
- P2-07 結果確定/反映待ち
- P2-08 ランキング
- P2-09 統計/履歴
- P2-10 ペナルティ通知
- P2-11 設定/プロフィール
- P2-12 エラー/メンテ

---

## 2. 画面詳細

### P2-01 ログイン/登録
- 目的: ユーザー認証
- 主なUI: フェーズ1同等

### P2-02 ホーム（インキュー/ステータス）
- 目的: キュー参加と待機状況表示
- 主なUI:
  - レート表示（現在のElo）
  - インキューボタン
  - 待機人数/経過時間
- 状態:
  - 未参加 / キュー中 / マッチ成立待ち
- アクション:
  - インキュー開始/停止

### P2-03 マッチ成立
- 目的: マッチ確定の通知とドラフト開始導線
- 主なUI:
  - チーム分け結果
  - 先攻/後攻表示
- 状態: 受理待ち / 参加確定

### P2-04 ドラフト（BAN/PICK）
- 目的: BAN/PICKを進行
- 主なUI/状態/アクション: フェーズ1同等

### P2-05 ロビーID入力/共有
- 目的: ロビーID共有
- 主なUI/状態/アクション: フェーズ1同等

### P2-06 試合結果入力
- 目的: 試合結果の自己申告
- 主なUI:
  - 勝ち/負け/無効の選択
  - 送信ボタン
  - 投票締切タイマー（5分以内）
- 状態:
  - 未投票 / 投票済み / 締切
- アクション:
  - 結果送信

### P2-07 結果確定/反映待ち
- 目的: 多数決確定と反映待ちの表示
- 主なUI:
  - 現在の投票状況
  - 多数決結果
  - 反映予定時刻（バッチ）
- 状態:
  - 投票集計中 / 確定 / 反映済み

### P2-08 ランキング
- 目的: レートランキング閲覧
- 主なUI:
  - ランキング一覧
  - 自分の順位
- 状態:
  - 読み込み中 / 表示

### P2-09 統計/履歴
- 目的: 直近試合履歴とレート変動
- 主なUI:
  - 試合履歴リスト
  - レート変動履歴
- 状態:
  - 読み込み中 / 表示

### P2-10 ペナルティ通知
- 目的: ノーショー等のペナルティ表示
- 主なUI:
  - ペナルティ理由
  - 解除予定時刻
- 状態:
  - ペナルティ中 / 解除済み

### P2-11 設定/プロフィール
- 目的: 表示名や基本設定の変更
- 主なUI/アクション: フェーズ1同等

### P2-12 エラー/メンテ
- 目的: システムエラー時の案内
- 主なUI/アクション: フェーズ1同等

---

## 3. 画面遷移（概要）

- P2-01 → P2-02
- P2-02 → P2-03 → P2-04 → P2-05 → P2-06 → P2-07 → P2-02
- 任意: P2-02 → P2-08 / P2-09 / P2-10
- 例外: エラー時は P2-12
