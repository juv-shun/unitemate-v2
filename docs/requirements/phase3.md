# フェーズ3要件: レーティングシステム

## 1. マッチングロジック (レートベース)
* **マッチング抽出（レート順）**:
  * キュー内の待機ユーザーをレートが高い順に並べ替える。
  * 上から順に10人ずつ抽出し、グループ（マッチング）を作成する。
* **チーム分け**:
  * 抽出された10人を、両チームの戦力が均衡するように振り分ける（スネーク方式）。

## 2. レート計算・更新
* **初期レート**: 1600
* **計算アルゴリズム**: Elo Rating System (K=32)
* **変動ロジック (順位相対評価)**:
  * レート計算は「チーム内でのレート順位が同じ相手プレイヤー」との比較で行う。
  * 例: 自チームでレートが2番目に高いプレイヤーは、相手チームでレートが2番目に高いプレイヤーと比較してElo計算を行う。
  * これを1位vs1位、2位vs2位... 5位vs5位の組み合わせで個別に計算し、各プレイヤーのレート変動を決定する。
* **変動タイミング**: フェーズ2の「勝敗確定」処理完了時
