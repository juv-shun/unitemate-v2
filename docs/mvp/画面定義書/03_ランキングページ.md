# 画面定義書 (ランキング)

## 概要

ユーザのレートによるランキング表示画面

## ログイン

不要

## 画面項目

| 項目名           | 説明                   |
| ---------------- | ---------------------- |
| ユーザーアイコン | Discord のアバター画像 |
| トレーナー名     |                        |
| Twitter(X) ID    |                        |
| レート           |                        |
| 勝率             |                        |

## ランキング表示ロジック

最終試合日時が現在日時より 7 日以上前のユーザーを条件にレートが高い順に 100 名表示。
(DB からは 150 名取得し、バックエンドアプリケーションコードにより、最終試合日時が 7 日以上前のユーザーを除外する)
