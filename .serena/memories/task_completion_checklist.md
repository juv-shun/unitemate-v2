# タスク完了時のチェックリスト

## コード修正後の確認事項

### 1. コード品質チェック
```bash
# ESLintでエラーがないか確認
pnpm lint

# Biomeでフォーマットとlintを実行
pnpm exec biome check --write .
```

### 2. TypeScript型チェック
```bash
# ビルドして型エラーがないか確認
pnpm build
```

### 3. 開発サーバーでの動作確認
```bash
# 開発サーバーを起動して動作確認
pnpm dev
```

### 4. ビルド確認
```bash
# 本番ビルドが成功するか確認
pnpm build

# ビルド後のプレビュー
pnpm preview
```

## コミット前の確認
- [ ] ESLintエラーなし
- [ ] TypeScriptコンパイルエラーなし
- [ ] Biomeフォーマット適用済み
- [ ] 開発サーバーで動作確認済み
- [ ] 本番ビルド成功

## デプロイ前の確認
- [ ] 上記すべての確認完了
- [ ] `.env`ファイルの設定確認（本番用の設定）
- [ ] Firebaseプロジェクト設定確認（`.firebaserc`）
- [ ] ビルド成果物（`dist/`）の確認

## 注意事項
- **ユーザーからの明示的な指示がない限り、自動的にコミットは実行しない**
- Gitコミットはユーザーが「コミットして」と明示的に指示した場合のみ実行
- ruffの実行は指定がない限り実行しない（Python用のため、本プロジェクトでは非該当）
